# MRAViewer: Machine Readable Appendix Viewer

## Abstract

MRAViewerはMRA dataの内容をブラウザーで表示するアプリケーションである。  

## Requirements

Web Browser  

## Installation

ファイルをダウンロードする。  

## Launch

htmlフォルダ内のindex.htmlをChromeなどのWeb Browserで開く。
