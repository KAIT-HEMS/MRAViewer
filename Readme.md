# MRAViewer: Machine Readable Appendix Viewer

2020.11.06  

## Abstract

MRAViewerはMRA dataの内容をブラウザーで表示するアプリケーションである。  

## Requirements

Web Browser  

## Installation

ファイルをダウンロードする。  
Download the files.

## Launch

htmlフォルダ内のindex.htmlをChromeなどのWeb Browserで開く。
Open index.html in html folder with a web browser such as Chrome.
